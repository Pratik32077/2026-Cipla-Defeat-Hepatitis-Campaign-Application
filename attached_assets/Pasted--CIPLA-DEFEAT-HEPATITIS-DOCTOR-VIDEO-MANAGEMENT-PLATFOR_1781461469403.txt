# CIPLA – DEFEAT HEPATITIS DOCTOR VIDEO MANAGEMENT PLATFORM

## PROJECT OVERVIEW

Build a production-ready, enterprise-grade Healthcare Campaign Management Platform for CIPLA's "Defeat Hepatitis" campaign.

The platform will be used to manage approximately 7000+ doctors through multiple managers across India.

The application must be:

* Fully Responsive
* Mobile First
* Progressive Web App Ready
* Optimized for Desktop
* Optimized for Tablet
* Optimized for Android Devices
* Optimized for iOS Devices
* Fast Loading
* Enterprise Grade
* Clean Healthcare UI
* Modern Dashboard Design
* Production Ready Architecture

The application must work smoothly across:

* Chrome
* Safari
* Edge
* Firefox
* Android Browser
* iPhone Safari
* iPad
* Windows
* macOS
